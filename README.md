## 📦 Instalation

You can use your favorite plugin manager for this. Here is an example with
packer.nvim:

#### packer.nvim

```lua
use({
    "zanglg/nova.nvim",
    config = function()
        require("nova").setup()

        -- load colorscheme
        vim.cmd([[colorscheme nova]])
    end,
})
```
